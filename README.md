# 🚢 Exploratory Data Analysis (EDA) on Titanic Dataset

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to understand its structure, identify missing values, detect outliers, analyze feature relationships, and uncover patterns related to passenger survival. Various statistical techniques and visualizations are used to derive meaningful insights before machine learning model development.

## 🎯 Objective

* Understand the dataset using descriptive statistics.
* Explore numerical and categorical features.
* Identify missing values and outliers.
* Analyze relationships between features.
* Draw insights through data visualization.

## 🛠️ Tools & Libraries

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

## 📂 Dataset

* **Dataset:** Titanic Dataset
* **Format:** CSV

## 📊 Exploratory Analysis Performed

* Loaded and explored the dataset
* Checked dataset shape, data types, and missing values
* Generated summary statistics
* Visualized numerical features using histograms
* Detected outliers using boxplots
* Created count plots for categorical variables
* Analyzed feature relationships using pairplots
* Generated a correlation heatmap
* Used Plotly for interactive visualizations
* Identified patterns, trends, and anomalies

## 📈 Key Insights

* The dataset contains missing values in the **Age**, **Cabin**, and **Embarked** columns.
* The **Fare** feature is positively skewed and contains several outliers.
* Female passengers had a higher survival rate than male passengers.
* First-class passengers had better survival chances than those in lower classes.
* Most passengers were between 20 and 40 years of age.
* No strong multicollinearity was observed among the numerical features.

## ▶️ How to Run

1. Open the notebook in Google Colab.
2. Upload the `Titanic-Dataset.csv` file.
3. Run all cells sequentially.
4. Review the generated statistics, visualizations, and insights.

## 📌 Outcome

This project demonstrates how Exploratory Data Analysis (EDA) helps understand a dataset, identify data quality issues, visualize feature distributions, and generate insights that support effective data preprocessing and machine learning model development.
